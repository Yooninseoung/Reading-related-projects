전자책을 이용해 독서를 하는데 플랫폼마다 제공해주는 전자책이 다르다.
내가 원하는 도서를 플랫폼에 일일이 들어가 검색하는 것은 매우 불편하다.
이 문제를 해결하고자 독서 관련 웹 어플리케이션을 만들었다. 
또한 독서와 관련된 웹 어플리케이션이기에 독후감을 작성할 수 있고 공공 데이터 포털을 이용해 도서관 사서 추천 도서를 사용자에게 제공할 것이다.


**기능 설명**
- 로그인 : 사용자는 로그인해야만 서비스를 이용할 수 있다.
- 회원 가입 : 사용자의 정보를 등록하여 회원 가입을 할 수 있다.
- 도서 추천 : 공공데이터 포털에서 제공하는 문화체육관광부_국립세종도서관 사서 추천 도서 6권을 사용자에게 제공해준다.
- 도서 검색 : 도서 이름을 검색창에 입력하면 여러 플랫폼의 검색 결과로 이동할 수 있다.
- 독서 기록 : 글 제목과 도서 제목, 내용 등을 기록, 수정, 삭제할 수 있다.
